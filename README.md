# Fortnite Zen Script Website

Premium Cronus Zen scripts for Fortnite. This is the official website for the **Titan Fortnite Zen Script**.

## Features

- **350+ AI Aim Assist Boosters** - Advanced aim assistance with Movement Prediction AI
- **Perfect Anti Recoil** - 100+ anti-recoil boosters for laser-accurate shots
- **Smart L2/R2 Adaptive Aim Assist** - AI that adapts to your playstyle
- **Controller Overclock** - Eliminate input delay for faster response
- **Lightning-Fast Pickup Macro 2.0** - 75% faster than competitors
- **Free Regular Updates** - Always compatible with the latest Fortnite patches

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- React Router DOM

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
